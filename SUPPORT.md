# Support

## Bugs

File a bug report using the
[bug report template](https://github.com/k2m30/a9s/issues/new?template=bug_report.yml).

Include your a9s version (`a9s --version`), operating system, and steps to
reproduce.

## Feature Requests

Submit a feature request using the
[feature request template](https://github.com/k2m30/a9s/issues/new?template=feature_request.yml).

## Questions and Discussion

For questions, ideas, and general discussion, use
[GitHub Discussions](https://github.com/k2m30/a9s/discussions).

## Security Issues

See [SECURITY.md](SECURITY.md) for how to report vulnerabilities. Do not open
public issues for security bugs.

## Contributing

See [CONTRIBUTING.md](CONTRIBUTING.md) for development setup and guidelines.
