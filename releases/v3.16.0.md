## New Features

- **IAM identity view** — press `i` to see AWS account, caller identity, and session info; header shows account badge and role/user name (#77)
- **EC2 Spot vs On-Demand** — new lifecycle column shows "spot" or "on-demand" with filter support (#72)

## Fixes

- **Name column first** — 14 resource types reordered so Name is the first column (sg, vpc, subnet, rtb, nat, igw, eip, vpce, tgw, eni, r53, cf, apigw, efs) (#23)
- Header identity badge gracefully drops on narrow terminals instead of wrapping
