## New Child View

- **Load Balancers → Listeners** (`Enter`): Drill into any ALB/NLB to see its listeners — port, protocol, default action (forward/redirect/fixed-response), target group, SSL policy, and certificate. Pagination via Marker/NextMarker, capped at 200 items. (#37)

## Details

### Listener columns
| Column | Source | Description |
|--------|--------|-------------|
| Port | `path: Port` | Listener port (443, 80, 8443, etc.) |
| Protocol | `path: Protocol` | HTTPS, HTTP, TCP, etc. |
| Action | `key: default_action_type` | forward, redirect, fixed-response |
| Target | `key: default_action_target` | TG name, redirect URL, or status code |
| SSL Policy | `path: SslPolicy` | e.g., ELBSecurityPolicy-TLS13-1-2-2021-06 |
| Certificate | `key: certificate_short` | ACM certificate ID (shortened from ARN) |
