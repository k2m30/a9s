# v3.28.0 — Related-Views Infrastructure

The foundational architecture for cross-resource navigation. This release adds the type system, message contracts, and UI components that all per-resource related views build on.

## What's New

**Two-column detail view** — Detail view now supports a right panel showing related resource counts and navigation links (#198).

**Type system** — `RelatedDef`, `NavigableField`, and `RelatedChecker` in `resource/related.go` provide a data-driven way to declare relationships between resource types.

**Message-driven dispatch** — `RelatedCheckResultMsg` and `RelatedNavigateMsg` handle async checker results and cross-resource navigation without any view-to-view coupling.

**Cache integration** — Checkers read from already-loaded resource lists before falling back to live API calls, minimizing AWS API usage.

## Key Bindings

| Key | Action |
|-----|--------|
| `r` | Toggle related panel |
| `Tab` | Switch between left/right columns |
| `Enter` | Navigate to related resource |
