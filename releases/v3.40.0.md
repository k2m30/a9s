## Detail View — Proper YAML Hierarchy

Nested structures in the detail view (arrays, maps, sub-objects) now render with the same hierarchical indentation as the YAML view. Security groups, block device mappings, IAM policy documents, and other complex fields are finally readable at a glance.

### What changed

- Sub-fields render with level-aware indentation instead of flat 5-space indent
- Array items show YAML list markers (`- `) matching the YAML view's structure
- Navigable fields within lists preserve their dash prefix for correct alignment
- Search queries match detail rows regardless of cursor position

### Test suite speedup

`make test` now runs in **4 seconds** (was 32s). The `-race` detector moved to `make test-race` for pre-push validation — same coverage, faster inner loop.
