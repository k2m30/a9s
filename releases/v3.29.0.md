# v3.29.0 — Related Resource Views for 23 Resource Types

Press `r` on any detail view to see what's connected. a9s now shows cross-resource relationships with live availability counts and one-key navigation.

## What's New

**Related resource panel** — The detail view now has a right column showing related resources. Each row displays a count (fetched asynchronously from cache or live API) and supports `Enter` to navigate directly to the filtered list.

**23 resource types** with full related-view support:

| Category | Resource Types |
|----------|---------------|
| Compute | EC2, ASG, Elastic Beanstalk |
| Containers | ECS Clusters, ECR Repositories |
| Databases | RDS, DocumentDB, DocumentDB Snapshots, DynamoDB |
| Storage | EBS Volumes, EBS Snapshots, AMIs |
| Networking | CloudFront, API Gateway, ACM |
| Monitoring | CloudWatch Alarms, CloudTrail Events |
| CI/CD | CloudFormation, CodeBuild, CodeArtifact |
| Messaging | EventBridge Rules |

**Navigable fields** — Fields like VpcId, SubnetId, ImageId, and KmsKeyId are highlighted in the detail view left column. Press `Enter` on a highlighted field to jump directly to that resource.

## Key Bindings

| Key | Action |
|-----|--------|
| `r` | Toggle related panel on/off |
| `Tab` | Switch focus between left and right columns |
| `Enter` | Navigate to selected related resource or navigable field |
| `Esc` | Close related panel / go back |
