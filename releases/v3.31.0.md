# v3.31.0 — Related Views for All 66 Resource Types

Every resource type now has a related-views panel. Press `r` on any detail view to see cross-resource relationships.

## What's New

**Complete coverage** — All 66 resource types have real related-view checkers. Zero nil stubs remain. A gate test (`TestGolden_LiveCheckerCompleteness`) fails the build if any nil checkers are introduced.

**43 new resource types** added since v3.29.0 (which had the first 23):
- Networking: VPC, Subnets, Security Groups, ELB, Target Groups, NAT, IGW, Route Tables, Transit Gateways, VPC Endpoints, Elastic IPs, ENI, WAF
- Security: IAM Roles, IAM Users, IAM Groups, IAM Policies, KMS, Secrets Manager, SSM Parameters
- Messaging: SQS, SNS, SNS Subscriptions, Kinesis, Step Functions, MSK, SES, Glue
- Databases: RDS Snapshots, Redis, Redshift, OpenSearch, DocDB Snapshots
- Storage: S3, EFS, Backup
- Containers: ECS Services, ECS Tasks, EKS
- CI/CD: CodePipeline, Athena
- Monitoring: Log Groups, CloudTrail

## Bug Fixes

- SNS Subscriptions no longer shows "(0+)" on accounts with zero subscriptions
- Secrets Manager and SSM Parameters now fetch 50 items per page instead of ~10
- Lowercase `m` triggers load-more (previously only `M`)

## Code Quality

- `interface{}` → `any` across 95 files
- `slices.Contains` replaces manual loops
- `strings.Cut` replaces `strings.Index`
