## a9s v3.5.0

### Windows Support

a9s now builds and releases for **Windows** (amd64 and arm64), in addition to macOS and Linux. (#55)

- **Scoop package manager** -- install on Windows with `scoop bucket add a9s https://github.com/k2m30/scoop-a9s.git && scoop install a9s`
- **Direct download** -- `.zip` archives available on the [releases page](https://github.com/k2m30/a9s/releases/latest)
- **Go install** -- `go install github.com/k2m30/a9s/v3/cmd/a9s@latest` (already cross-platform)
- **CI** -- Windows added to the test matrix alongside Ubuntu and macOS

> **Note:** Downloaded Windows binaries may trigger a Microsoft Defender SmartScreen warning because they are not yet code-signed. Install via Scoop to avoid this, or click "More info" → "Run anyway". Code signing is tracked in #76.

### Install

```sh
# macOS / Linux
brew install k2m30/a9s/a9s

# Windows
scoop bucket add a9s https://github.com/k2m30/scoop-a9s.git
scoop install a9s

# Any platform
go install github.com/k2m30/a9s/v3/cmd/a9s@v3.5.0
```
