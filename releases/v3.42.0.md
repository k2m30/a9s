## View Switching, Column Sort, and Error Log

### Inter-view navigation (#269)

Detail, YAML, and JSON views are now fully inter-navigable. Press `d`, `y`, or `J` from any of them to switch directly — no need to go back first. The current view is replaced in-place, so `Esc` still returns to the parent detail view.

### Sort by column position (#267)

Press `1`–`9` or `0` to sort the resource list by that column (1 = first, 0 = tenth). Press the same key again to toggle between ascending and descending. A sort indicator (▲/▼) appears in the column header.

### Error log (#268)

Press `!` to open a session error log showing all AWS API errors with timestamps. Useful for diagnosing connectivity or permission issues without losing your place.
