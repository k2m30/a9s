## Navigation, Tags, and JSON Pretty-Printing

### `:root` / `:main` command (#258)

Jump back to the main menu from any view depth with a single command — no more repeated `Escape` presses. Tab completion included.

All help screens now show a **COMMANDS** section listing every available colon command.

### Flat tag rendering (#210)

AWS tags in detail views render as clean `Key: Value` pairs instead of the verbose SDK struct format. Tags with extra metadata (e.g., Auto Scaling Group `PropagateAtLaunch`) are left unflattened so no information is lost.

### JSON pretty-printing (#262)

Field values containing embedded JSON (policy documents, environment variables, event patterns) are automatically detected and expanded as indented sub-fields in the detail view. YAML and JSON views are unaffected.

Secret and parameter values revealed with `x` now render as syntax-colored indented JSON, making structured secrets readable at a glance. Copy (`c`) still returns the raw original value.
