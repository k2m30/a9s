## a9s v3.0.1

### Fixes

- **Version info for `go install`** -- `a9s --version` now resolves commit hash and build date from Go's embedded VCS metadata when ldflags aren't set. Previously showed `commit: none, built: unknown`.
- **Docker semver tags** -- images are now tagged with `v3`, `v3.0`, `v3.0.1`, and `latest` (previously only the full version and `latest`).
- **Release pipeline** -- fixed OOM during cross-compilation by limiting GoReleaser parallelism; Docker build runs independently of GoReleaser.
