## a9s v3.4.0

### New Child Views: CloudFormation Stacks

Two new child views for CloudFormation Stacks -- the second multi-child parent in a9s:

- **CFN Stacks → Stack Events** (`Enter`) -- Real-time deployment timeline showing which resources are being created/updated, which failed, and why. Paginated via NextToken, capped at 200 events. The first thing to check during an active deployment or rollback.
- **CFN Stacks → Stack Resources** (`r`) -- Logical-to-physical resource mapping with status and drift detection. Shows what the stack manages, which resource is stuck, and whether anything has drifted. Paginated to collect all resources.

### How it works

CFN Stacks is the second **multi-child parent** (after ECS Services):
- `Enter` = Stack Events (what happened during deployment?)
- `r` = Stack Resources (what does this stack manage?)

Both views use the existing CloudFormation client (Pattern B: client reuse). Status coloring shows COMPLETE in green, FAILED/ROLLBACK in red, and IN_PROGRESS in yellow. Stack Resources also highlights MODIFIED drift status in red.
