## Bug Fixes

- **Error flash auto-dismiss** — flash messages now disappear after 5 seconds instead of persisting forever (#86)
- **Long error truncation** — long error messages truncated in header to prevent multi-line wrapping that pushed content off-screen (#84)
- **KMS graceful error handling** — fetcher now skips keys that can't be described (e.g. permission denied) instead of failing entirely (#85)
- **DynamoDB graceful error handling** — fetcher now skips tables that can't be described instead of failing entirely
