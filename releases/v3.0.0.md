## a9s v3.0.0 -- First Public Release

Terminal UI for AWS resource management. Browse 62 resource types across 12 categories with vim-style navigation, real-time filtering, and drill-down views.

### Highlights

- **62 AWS resource types** across Compute, Networking, Databases, Security, CI/CD, and more
- **Child view architecture** -- drill into S3 objects, Route53 records, and 21 more planned parent-child views via a generic, data-driven system
- **Tokyo Night Dark theme** with status-aware row coloring
- **Configurable columns** via `~/.a9s/views.yaml`
- **Multiple install methods** -- Homebrew, go install, Docker, binary downloads

### Install

```sh
brew install k2m30/a9s/a9s
# or
go install github.com/k2m30/a9s/v3/cmd/a9s@v3.0.0
# or
docker run --rm -it ghcr.io/k2m30/a9s:v3.0.0 --demo
```
