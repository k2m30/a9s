## a9s v3.1.0

### New Child Views

- **Log Groups → Log Streams** -- Press Enter on a CloudWatch Log Group to drill into its log streams. Streams are sorted by most recent event first. Shows stream name, last/first event timestamps, and stored size in human-readable format. Paginated via DescribeLogStreams API.
- **Log Streams → Log Events** -- Press Enter on a log stream to view its log events. Events are color-classified: ERROR/FATAL (red), WARN (yellow), REPORT (green), START/END (dim). This forms a 3-level drill-down chain: Log Groups → Log Streams → Log Events.

### Foundation

This release establishes the log viewer infrastructure that all cross-service log views (Lambda, ECS Container Logs, CodeBuild Build Logs) will reuse.
