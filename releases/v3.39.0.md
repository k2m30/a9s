## View Customization — Out of the Box

a9s now auto-creates per-resource view config files in `~/.a9s/views/` on first launch. Customize which columns appear in list views and which fields show in detail views by editing any YAML file — a9s never overwrites your edits.

A complete field reference (`~/.a9s/views_reference.yaml`) is also maintained automatically and stays up-to-date with each binary upgrade.

### New CLI flags

```sh
a9s --reset-views   # delete all view configs, regenerate defaults on next launch
a9s --reset-themes  # delete all theme files, regenerate defaults on next launch
```

Both flags prompt for confirmation before deleting.
