## New Child Views

- **CodeBuild Projects → Builds** (`Enter`): Drill into any project to see recent builds with status, duration, source version, and initiator. (#41)
- **CodeBuild Builds → Build Logs** (`Enter`): Full build output from CloudWatch Logs with phase/error highlighting. Build log status classification: ERROR (red), SUCCEEDED (green), IN_PROGRESS (blue) for quick visual scanning. `DrillCondition` shows "Build logs not available in CloudWatch" when logs are disabled. (#42)
