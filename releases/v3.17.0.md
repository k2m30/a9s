# v3.17.0 — Unified Pagination

All 62 top-level fetchers and 28 child fetchers now support pagination with load-more UX.

## What's New

**Unified pagination** — Every fetcher now reports truncation status. Paginated lists show a count indicator (e.g., "25+") and support `M` to load more (#83, #80).

**Retry infrastructure** — `RetryOnThrottle` handles AWS API rate limiting with exponential backoff.

**Demo mode pagination** — About 30% of resources demonstrate load-more UX with 5-item pages.

**65 QA stories** with full test coverage for the pagination feature.
