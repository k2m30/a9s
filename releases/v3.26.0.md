## Resource Cache, Cross-View Search, Throttle-Resilient Probes (#111, #89, #186)

### Resource Cache (#111)

Navigating away from a resource list (Esc) and re-entering no longer re-fetches from AWS. Loaded resources are preserved in a write-through cache that invalidates selectively on profile or region switch.

### Cross-View Search (#89)

Press `/` in any list, detail, or YAML view to search. Type to filter, `n`/`N` to jump between matches, with ANSI-aware highlighting that works across all view types. Enter on empty query deactivates search.

- 68 QA story tests covering activation, typing, highlighting, navigation, case sensitivity, edge cases, key interactions, resize, and reuse

### Throttle-Resilient Availability Probes (#186)

Availability probes now retry up to 3 times with exponential backoff when AWS returns `ThrottlingException`, instead of silently failing and marking resources as unavailable.
