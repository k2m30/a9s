# S3 Buckets (s3) — Related Resources

## Real-World Use Cases

**1. "Is this bucket exposed to the internet?"** You need to check the bucket policy for public access, the bucket's public access block settings, and any CloudFront distributions that serve it. A bucket can be private at the bucket level but public through a CloudFront OAI/OAC.

**2. "Which services write to this bucket?"** During cost analysis or data lineage investigation, you need to find all producers: ELB access logs, CloudTrail trail delivery, VPC flow logs, Athena query results, Glue job output, CodeBuild artifacts. Each is configured on the producer's side — the bucket has no idea who writes to it.

**3. "What happens when a file is uploaded?"** S3 event notifications trigger Lambda functions, SQS queues, or SNS topics. The notification configuration is on the bucket but is a separate API call from the bucket listing.

**4. "Is this bucket replicated?"** Replication configuration shows the destination bucket in another region or account. Important for DR verification.

## Reverse Relationships

| Related Resource | How to Find | Scenario | Priority |
|-----------------|-------------|----------|----------|
| CloudFront Distributions (cf) | Search CF distributions for origins where `DomainName` matches `{bucket}.s3.amazonaws.com`, `{bucket}.s3.{region}.amazonaws.com`, or `{bucket}.s3-website-{region}.amazonaws.com`. If a9s has CF data cached, search in-memory. | "Is this bucket served through a CDN?" Common for static websites and asset delivery. | P0 |
| CloudTrail Trails (trail) | Search trails for `S3BucketName` matching this bucket. If a9s has trail data cached, search in-memory. | "Does CloudTrail deliver logs to this bucket?" Critical for understanding the bucket's role in audit infrastructure. | P1 |
| ELB (elb) — Access Logs | Search ELBs whose attributes have `access_logs.s3.bucket` matching this bucket name. Requires `elbv2:DescribeLoadBalancerAttributes` per ELB, or search cached ELB data. | "Which load balancers write access logs here?" | P1 |
| VPC Flow Logs (not in a9s) | `ec2:DescribeFlowLogs` and filter for `LogDestinationType=s3` with `LogDestination` containing this bucket's ARN. | "Do VPC flow logs write here?" | P2 |
| Athena Workgroups (athena) | Search workgroup configs for `ResultConfiguration.OutputLocation` containing this bucket name. | "Does Athena write query results here?" | P2 |
| CloudFormation Stacks (cfn) | Check for `aws:cloudformation:stack-name` tag. | "Which stack manages this bucket?" | P2 |

## Algorithmic Relationships

| Related Resource | Algorithm | Scenario | Priority |
|-----------------|-----------|----------|----------|
| Lambda / SQS / SNS — Event Notifications | `s3:GetBucketNotificationConfiguration` returns `LambdaFunctionConfigurations[]`, `QueueConfigurations[]`, `TopicConfigurations[]` — each with the target ARN and event filter (e.g., `s3:ObjectCreated:*`). | "What triggers when files are uploaded?" Critical for understanding data pipelines. Uploading a file may trigger a Lambda that processes it, or an SQS message for async processing. | P0 |
| S3 Replication Destination (s3) | `s3:GetBucketReplication` returns `Rules[].Destination.Bucket` — the destination bucket ARN. | "Is this bucket replicated to another region?" DR verification. | P1 |
| KMS Key (kms) | `s3:GetBucketEncryption` returns `Rules[].ApplyServerSideEncryptionByDefault.KMSMasterKeyID`. If set, all objects are encrypted with this KMS key. | "What key encrypts this bucket's data?" Security audit and access control analysis. | P1 |
| IAM Roles/Users — Bucket Policy | `s3:GetBucketPolicy` returns the policy JSON. Parse `Principal` fields to identify IAM roles, users, accounts, and services with access. Also check for `Condition` keys like `aws:sourceVpce` (VPC endpoint restrictions) or `aws:sourceIp` (IP restrictions). | "Who has access to this bucket?" Security audit — especially for buckets with sensitive data. | P1 |
| Lifecycle Rules (not a resource) | `s3:GetBucketLifecycleConfiguration` shows transition and expiration rules. Not a related resource but answers "why did my objects disappear?" and "why is this bucket so expensive?" (objects not transitioning to cheaper storage). | "Where did my data go?" or "Why isn't data aging into Glacier?" | P2 |

## CloudTrail Events (T key)

| Event Name | Why Engineers Search For It |
|-----------|---------------------------|
| DeleteBucket | "Who deleted this bucket and all its data?" Buckets must be empty to delete, so this is a deliberate action. |
| PutBucketPolicy | "Who changed the bucket policy?" A permissive policy change can expose data. The most security-sensitive S3 event. |
| PutBucketPublicAccessBlock / DeletePublicAccessBlock | "Who changed public access settings?" Making a bucket public is a critical security event. |
